	СТАТИСТИКА
Prec = TP/(TP+FP) - не взять лишнего
Recall = TP/(TP+FN) - не пропустить нужного
F1 мера предсказаний 2 * PREC *RECALL/(RECALL+PREC)
	Основные понятия
Генеральная совокупность (ГС) - все объекты относительно которых хочется сделать вывод
Выборка - часть ГС
Стратифициорванная выборка (СТВ) - разбиение на страты и равномерны выбор из каждой
Групповая выборка (ГВ) - выбор только из определеной страты
	Переменные
Количественные - непрерывные и дискретные
Номинативные (пол)
Ранговые
	Описательные характеристики выборки
Мера центральной тенденции - величина принимаемых значений
Мера изменчивости - размер промежутока принимаемых значений
Мода - самое частое значение
Медиана - значение посередине упорядоченного множества
Размах - заница макс и мин
Дисперсия - среднее квадратов отклонений от среднего для каждого значения
	Центральная теорем 
Сигма - корень из дисперсии для всей ГС, для выборки sd
Для сигмы | sig =  sumal(xi-xs)/n |, для | sd =  sumal(xi-xs)/n-1 |
Квантили - точки разделяющие на три равные группы в упорядоченном множестве
Стандартизация - приведение к виду xs = 0 и D = 1, вычесть среднее и делать на sd
Выброчное среднее (ВС) - среднее значение выборки
Среднее ВС-них равно среднему ГС, его стандартное отклонение se = sigma/sqrt(n), n - колво элементов выборок
для n>30, se = sd/sqrt(n)
Доверительный интервал - т.к. мы знаем что в с 95% шансом среднее нашей выборки лежит
в +- 1.96 se от среднего ГС, то с тем же шансом среднее ГС, лежит в +- 1.96 se интервале нашего среднего 
Для т распределения все тоже самое но график дургой
Для двух выборок x1-x2 = a, se = sqrt(sd1^2/n1+sd2^2/n2), степень свободы df = n1+n2-2
	Однофакторный дисперсионный анализ
| Xs = sumal(xij)/m*n | - i - группа, j - номер в группе, m -колво групп, n - колво элементов в группе
SST - sumal(xij-Xs)^2
df - m*n-1
SSB - сумма квадратов межгруп = sumal(Xs - Xsi)^2*ni, d = n - 1
SSD - сумма квадратов внутригрупповая = sumal(xji-Xsi)^2"по i,j", d = m - n
F критерий = SSB/(-1)
	Множественные сравнения
Если мы берем n выборок по m значений, и хотим найти хотя бы одно значимое отличие для подтверждения
различий, то для пары групп, критерий прохождения p надо делить на количество попарных сравнений = n*(n-1)
